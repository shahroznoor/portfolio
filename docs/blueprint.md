# **App Name**: ProfileCanvas

## Core Features:

- Profile Import: Automatically fetch and display professional details (work experience, education, skills, summary) from the provided LinkedIn profile.
- AI-Powered Content Creation: Employ an AI tool to analyze LinkedIn profile content and auto-generate compelling personal statements for the 'About Me' and 'Summary' sections.
- Customizable Sections: Allow users to manually edit and customize all sections of the portfolio to add personal flair and additional information beyond their LinkedIn profile.
- Project Showcase: Enable the uploading and showcasing of projects and work samples (images, videos, PDFs) to highlight practical experience and achievements.
- Contact Section: Implement a contact form for visitors to easily reach out, enhancing networking opportunities.
- Shareable URL: Generate a unique URL that can be shared across all social and professional platforms.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism, intelligence, and trustworthiness.
- Background color: Light Grey (#F0F2F5), providing a neutral, distraction-free backdrop to keep focus on the content.
- Accent color: Soft Purple (#9C27B0), positioned approximately 30 degrees counterclockwise from the primary on the color wheel, adds a touch of sophistication without being overbearing.
- Body text: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif. This pairing balances modern readability and techy sophistication.
- Use simple, professional line icons related to skills and expertise to create visual interest.
- Use a clean, single-page layout with clearly defined sections to enhance readability.
- Subtle scrolling animations that highlight different parts of the page as the user navigates down.
